<h2 align="center">Prism (0.0.1)</h2>
<h3>Introductie</h3>
Prism is een taal qua structuur gelijkend op C. Dit is voornamelijk vanwege control-flow.


<h3>Variabelen</h3>
In Prism hebben variabelen vijf eigenschappen: naam, waarde, scope, type en bewerkbaarheid

<h4>Naam</h3>
De naam van een variabele wordt simpelweg aangegeven met het woord voor het is-teken.
```c
// |
//\/
variable = ...
```

<h4>Waarde</h3>
De waarde van de variabele is de waarde _achter_ het is-teken
